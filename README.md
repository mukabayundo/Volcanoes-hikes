# Volcanoes-hikes
This is a zero-dependency, single-file project — no build step, no npm, no bundler.

## 🚀 Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/your-mukabayundo/rwanda-volcanoes-explorer.git
   cd rwanda-volcanoes-explorer
```

2. **Open in your browser**
```bash
   open index.html
```
   Or drag `index.html` into any modern browser.

---

## 🛠️ Technologies Used

- **HTML5** — semantic structure
- **Tailwind CSS** (CDN) — utility-first styling and dark mode via `darkMode: 'class'`
- **Vanilla JavaScript** — all interactivity and DOM manipulation, no libraries

---

## 📚 DOM Concepts Demonstrated

- `createElement`, `appendChild`, `remove()` for dynamic list building
- `addEventListener` for clicks, keyboard (`Enter` key), and input events
- `dataset` attributes (`data-name`, `data-locked`) for element metadata
- `classList.toggle` / `classList.add` / `classList.remove` for UI state
- `querySelectorAll` + array iteration for filtering and batch updates
- `scrollIntoView` for smooth auto-scroll on activity selection
- Conditional rendering with `style.display` and `hidden` class toggling

## 🌍 About the Virunga Massif

The Virunga chain spans Rwanda, Uganda, and the DRC and includes 8 volcanoes.
Rwanda's Volcanoes National Park is home to approximately one third of the world's
remaining mountain gorillas and was the site of Dian Fossey's famous gorilla research at Karisoke.

**Volcanoes featured in this app:**

| Volcano | Elevation |
|---|---|
| Mount Karisimbi | 4,507 m |
| Mount Muhabura | 4,127 m |
| Mount Bisoke | 3,711 m |
| Mount Sabyinyo | 3,634 m |
| Mount Gahinga | 3,474 m |
